var可以定义局部或者全局变量，下面的代码定义了两个整形的变量：
var x,y:int 
我们还可以用换行的方式，定义一系列的变量，下面的代码定义了两个整形的变量和两个字符串类型的变量：
var 
    x , y : int
    i , z :  string
如果要在声明的时候就赋值，那么可以写成这样的形式，下面的代码定义了两个整型变量，并赋值为3.
var x , y = 3
在上面这行代码中，并没有为变量x和y设置变量的类型，这是因为程序编译的时候，编译器可以根据这两个变量的值推断出这两个变量的类型，所以开发人员可以不用为这样的变量设置变量类型。这就是Nim的类型推断。
Nim语言和其他语言一样，也有常量。Nim语言使用const关键字定义常量，const关键字和var关键字在使用方法上类似，也可以换行批量定义：
const 
    x = 4
    y = x+5
    z = "allen"
需要注意的是：
?和其他语言定义的常量一样，const定义的常量不能被修改。
?因为编译器会把所有const变量换成他所对应的值，所以变量对应的值是表达式的话，在编译期一定要能计算出这个表达式的值来才行。
Nim语言还可以使用let关键字定义变量，用let关键字定义的变量，赋值后也不能被修改，但用let定义的变量，可以在运行期赋值。请看下面两行代码：
const input = readLine(stdin) # Error: 运行期的值不能赋给const变量
let input = readLine(stdin)   # works：运行期的值可以赋给let定义的变量
